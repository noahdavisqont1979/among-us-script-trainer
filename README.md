## What This Is

Among Us Trainer is a game utility built around in-match helpers and control options for the HTML-based target. Its focus is on features such as ESP wallhack, impostor reveal, teleport, kill-anywhere style control, speed manipulation, and cosmetic unlocks, giving users a configurable toolset to load and use during play.

This repository is organized as a trainer-style script utility, not as a broad standalone application. Ongoing work is expected to keep the feature set available, preserve usability for the target platform, and stay aligned with the current game build and browser-based runtime.
